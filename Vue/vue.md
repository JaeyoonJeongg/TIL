# Vue.js

1. 뷰 시작하기

    - 큰 프로젝트에서는 CDN 보다는 NPM 사용

    - 터미널에서 Vue CLI 설치

      ``` 
      npm install -g @vue/cli
      vue --version
      ```

    - 뷰 프로젝트 시작하기

      ``` 
      vue create vue3-test //vue3-test 라는 폴더(프로젝트명) 만듬
      cd vue3-test
      ```

    - vsCode, intelliJ 등 에서 프로젝트 열기

    - package.json 에서 프로젝트 상세정보 확인 가능
